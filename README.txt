This is the project built for the second delivery for DP2.
This project has different functionalities depending on the role of the user. 
	- Anonymous can list and create a shout, it also can list and show some tasks and workplans.
	- Managers can list public tasks, as well as, its own tasks and own workplans. This users can create tasks and workplans for themselves.
	- Administrators can list all user-accounts, tasks and workplans' dashboards, lista, modify and delete spamwords and threshold, populate the database and shut the system down.
